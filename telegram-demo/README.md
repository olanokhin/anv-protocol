# Telegram Demo — PoC Stage 2

Planned implementation: ANV authorization type indicator
in a Telegram Web client fork (tweb).

## Scope

- Pyrogram userbot as AI agent (initiates calls with ANV metadata)
- tweb fork with ANV badge on incoming call screen
- Three scenarios: SIGNED_AI, UNSIGNED warn, UNSIGNED blocked
