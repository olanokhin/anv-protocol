# Telegram Demo — PoC Stage 2

Planned implementation: ATA authorization type indicator
in a Telegram Web client fork (tweb).

## Scope

- Pyrogram userbot as AI agent (initiates calls with ATA metadata)
- tweb fork with ATA badge on incoming call screen
- Three scenarios: SIGNED_AI, UNSIGNED warn, UNSIGNED blocked
